認證說明

  • 認證網站
  • 認證內容
    • 題組
      • 單一科目 6 個專案
      • 各專案 5~7 個任務不等
      • 考試題庫一共二個版本
    • 考試時間及分數
      • 時間 50 鐘
      • 滿分 1000 分，及格 700 分
  • 認證分類
    • Associate 等級
      • Word Associate ( Exam MO-100 )
      • Excel Associate ( Exam MO-200 )
      • Powerpoint Associate ( Exam MO-300 )
      • outlook Associate ( Exam MO-400 )
      • 考取上述各科可以分別取得相關的證書，另外考取四科中其中三科可以取得 Microsoft Office Specialist - Associate 證書
    • Associate 證書

    • Expert 等級
      • Word Expert ( Exam MO-101 )
      • Excel Expert ( Exam MO-201 )
      • Access Expert ( Exam MO-500 )
      • 考取上述各科可以分別取得相關的證書，另外考取三科中其中二科可以取得 Microsoft Office Specialist - Expert 證書
    • Expert 證書

  • 考試完成
